Volunteer Manage System

C# Visual Studio HTML JavaScript SQLite

The Volunteer Scheduling System is a web-based platform designed to simplify volunteer management. Traditionally, volunteer coordination was done manually using Excel, which was time-consuming and prone to errors. Volunteers had to register through the manager, and scheduling events required multiple rounds of emails to confirm availability, often causing gaps or overlaps in the schedule.

This system allows volunteers to register online and select their preferred shifts, while managers can create event schedules, track volunteer hours, and send notifications automatically. The platform reduces coordination time, ensures accurate scheduling, and improves overall efficiency.
