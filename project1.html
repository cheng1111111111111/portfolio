Collection Game Details

C# Visual Studio

This C# game implements the core mechanics of a classic single-player arcade game: the player moves using the arrow keys while enemies move autonomously around the map. The player must collect items to gain points, race against a 60-second timer, and avoid enemy collisions. Each new level increases difficulty by adding more enemies. The game ends when time runs out or when the player is hit, and it provides restart/exit options as well as a high-score notification system.
