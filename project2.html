Online Mini Card Game

JAVA Eclipse

The server gives each player a random “card” from 1 to 13, and both players reveal their card at the same time. The higher card wins, and the score updates automatically.

This game already includes the core flow of an online card game:

  • Dealing cards — the server gives each player a random number
  • Playing the card — players press the Reveal button
  • Comparing the cards — the server decides who wins
  • Updating the score — the result appears for both players
